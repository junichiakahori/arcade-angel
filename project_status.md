# Live2D Arc Viewer Project Handover

## 1. 現状のステータス
`arc_viewer.html` は現在、完全な動作状態にあります。`file://` プロトコルの制限を回避するため、すべての主要アセットを Base64 形式で埋め込み済みです。

## 2. 実施済みの主要な修正
- **Base64 アセットの埋め込み**:
  - `face`, `eyes`, `mouth` のパーツを `_b64.txt` から読み込み、HTML 内に直接ハードコードしました。これにより、ローカル環境で直接開いても画像が正常にロードされます。
- **肌の全自動レタッチ (Skin Warmth Filter)**:
  - `removePink` 関数内で、冷たい白を「人間らしい温かみのある肌色」にリアルタイム補正する処理を追加しました（輝度と色の距離ベース）。
- **口の比率と動きの最適化 (v5.1)**:
  - **ブラウザ自動検証**: エージェントによるDOMと描画の自動チェックを実施。閉じ口が鼻幅に対して広いことを特定。
  - **比率修正**: 閉じた時の横幅を `0.48` まで絞り、開いた時にダイナミックに広がるよう調整（scaleXの個別制御を導入）。
  - **顎運動**: 口の開閉に連動した「アゴの下がり（jaw drop）」アニメーションを実装。
- **トラッキングの平滑化**:
  - MediaPipe からのデータをスムージングし、カクつきを抑えました。
- **描画パースの強化**: 首を横に振った際の顔の厚み（圧縮効果）を `0.12` に強化。

## 3. 主要ファイル
- [arc_viewer.html](./arc_viewer.html): 現在の最新版。全アセット・全ロジックが集約されています。
- [index.html](./index.html): PixiJS と MediaPipe の基本連携が確認できるリファレンス用ファイル。
